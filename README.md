# devtrack

Um repositório criado para uso no aprendizado de node.

## Requisitos

- Node v19 ou acima
- CLI compatível

## Setup

Para instalar o projeto para seu próprio uso, basta clonar o repositório:

```
https://github.com/diego-rng/devtrack
```

Para iniciar a CLI, use o comando:

```
npm run start
```

Ou se quiser fazer mudanças no código e testá-las sem reiniciar a CLI, use:

```
npm run dev
```

Para encerrar o processo, pressione o número 4 no Menu ou use Ctrl + C, e verá isso na tela

```
>
Encerrando...
```
